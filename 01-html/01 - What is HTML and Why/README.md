# Lesson 01 - What is HTML and Why

---

# 📘 Introduction

In this lesson, we learned about:

# HTML

is:

> **HTML** stands for **HyperText Markup Language**. It is the standard markup language used to create and structure web pages.

---

# 🎯 Main Idea

HTML tells the browser what the content means.

It is used to structure text, links, images, and other content on a web page.

> HTML is not about how things look. It is about what things mean.

---

# 💡 Example

**A simple HTML page:**

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>01 - What is HTML and Why</title>
</head>

<body>
    <h1>What is html </h1>
    <p>HTML is not about how things look.
        It is about what things mean.
    </p>
    <a href="https://www.example.com">Visit Example</a>
    <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
</body>

</html>
```

---

# ✨ Basic HTML Structure

**✅ Main parts of an HTML document:**

**Document Type:**
```html
<!DOCTYPE html>
```

**HTML Root Element:**
```html
<html lang="en">
</html>
```

**Head Section:**
```html
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
```

**Body Section:**
```html
<body>
    <h1>Main Heading</h1>
    <p>Page content goes here.</p>
</body>
```

---

# 🧠 Rules of HTML

```text
✔️ HTML is a markup language, not a programming language
✔️ HTML structures the content of a web page
✔️ The browser reads HTML and displays the page
✔️ HTML elements usually have opening and closing tags
✔️ The visible content goes inside the body section
```

---

# 🔥 Important Rule

```text
HTML = Structure
CSS = Style
JavaScript = Behavior

HTML does not control the design by itself.
It gives meaning and structure to the page content.
```

---

# 🧩 HTML Checklist

| Part | Meaning | Example |
| ---- | ------- | ------- |
| Doctype | Defines HTML5 document | `<!DOCTYPE html>` |
| Root | Contains the whole page | `<html>` |
| Head | Page settings and information | `<head>` |
| Title | Browser tab title | `<title>` |
| Body | Visible page content | `<body>` |
| Heading | Main title in the page | `<h1>` |
| Paragraph | Text block | `<p>` |
| Link | Clickable URL | `<a>` |
| Image | Displays an image | `<img>` |

---

# ✅ Summary

| Concept | Meaning |
| ------- | ------- |
| HTML | HyperText Markup Language |
| Purpose | Structure web pages |
| Type | Markup language |
| Browser | Reads HTML and displays content |
| Works With | CSS and JavaScript |
